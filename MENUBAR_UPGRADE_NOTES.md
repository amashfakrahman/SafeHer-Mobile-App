# SafeHer Menu Bar Upgrade

This update adds a simple user-friendly menu tab to the SafeHer mobile app.

## What changed

- Replaced the bottom `Profile` tab with a clearer `Menu` tab.
- Added `frontend/src/screens/main/AppMenuScreen.js`.
- Grouped app features into easy sections:
  - Fast access
  - Safety tools
  - Account
- Added shortcut buttons for Home, Live Share, Contacts, and Report.
- Added menu rows for Incident Reports, Help Centers, Evidence Vault, Community, Profile & Settings, and Log out.
- Moved Profile into the root stack so it opens from the menu or Home settings button.
- Added a back button to the Profile screen when opened from the menu.

## Updated navigation

Bottom bar now shows:

1. Home
2. Contacts
3. Location
4. Reports
5. Menu

The Menu screen gives users one friendly place to find every app feature.
